Getting started
===============

The PRiSM tool runs using `python`. To help with setup, we use `make`. It is assumed you have the following installed: `python 3.11`, `pip`, and [`make`](https://cookiecutter-data-science.drivendata.org/using-the-template/#make-as-a-task-runner).

Open a terminal and navigate to the project folder (where this `README.md` is). Start by creating the python virtual environment by running.

```bash
make create_environment
```

To activate the virtual environment run:

On Windows: `.\prism_venv\Scripts\activate`

On Unix-like system: `.\prism_venv\bin\activate`

See `\notebooks\4.01-hpi-unos-modelling-full.ipynb` for an example runthrough of the PRiSM method (_to be replaced with a proper tutuorial notebook_).
